# Dr. Aksana Ismailbekova - Academic Portfolio Website

A modern, responsive portfolio website for Dr. Aksana Ismailbekova, Research Fellow at the Leibniz-Zentrum Moderner Orient (ZMO) in Berlin, Germany.

## Features

- **Modern Design**: Clean, professional academic portfolio design
- **Responsive Layout**: Optimized for desktop, tablet, and mobile devices
- **Smooth Navigation**: Single-page application with smooth scrolling
- **Interactive Elements**: Hover effects, animations, and mobile-friendly navigation
- **Contact Form**: Integrated contact form that opens user's email client
- **Academic Focus**: Showcases research, publications, and professional experience

## Sections

1. **Home**: Hero section with introduction and quick links
2. **About**: Detailed biography and educational background
3. **Research**: Research interests and professional experience
4. **Publications**: Books and selected academic articles
5. **Contact**: Contact information and inquiry form

## Deployment Options

### 1. GitHub Pages (Recommended - FREE)

**Advantages**: Free hosting, custom domain support, automatic HTTPS, easy updates via Git

**Steps**:
1. Create a new GitHub repository named `aksana-ismailbekova-portfolio`
2. Upload all files to the repository
3. Go to repository Settings → Pages
4. Select "Deploy from a branch" and choose `main` branch
5. Your site will be available at `https://yourusername.github.io/aksana-ismailbekova-portfolio`

**Cost**: FREE
**Custom Domain**: Supported (additional domain cost applies)

### 2. Netlify (Recommended - FREE with premium features)

**Advantages**: Free hosting, custom domain support, form handling, continuous deployment

**Steps**:
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop the project folder to Netlify
3. Your site will be deployed instantly with a random URL
4. Optionally connect to GitHub for automatic updates

**Cost**: FREE (up to 100GB bandwidth/month)
**Custom Domain**: Supported (additional domain cost applies)

### 3. Vercel (FREE)

**Advantages**: Fast global CDN, automatic HTTPS, great performance

**Steps**:
1. Go to [vercel.com](https://vercel.com)
2. Import from Git repository or upload files
3. Deploy with one click

**Cost**: FREE for personal projects
**Custom Domain**: Supported (additional domain cost applies)

### 4. Firebase Hosting (FREE tier available)

**Advantages**: Google's infrastructure, fast loading, SSL included

**Steps**:
1. Install Firebase CLI: `npm install -g firebase-tools`
2. Initialize Firebase in your project directory
3. Deploy using `firebase deploy`

**Cost**: FREE (up to 10GB storage, 1GB transfer/month)

## Local Development

To test the website locally:

1. Open terminal in the project directory
2. Start a simple HTTP server:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Python 2
   python -m SimpleHTTPServer 8000
   
   # Node.js (if you have it installed)
   npx http-server
   ```
3. Open browser and go to `http://localhost:8000`
